# Course 1 PreFlop - Poker Terms Glossary

A searchable glossary of poker terms and abbreviations used in the Course 1 PreFlop curriculum.

## Quick Start

```bash
npm install
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173).

```bash
npm run build
npm run preview
```

## About

Glossary of poker terminology including:
- Positions (UTG, EP, MP, HJ, CO, BTN, SB, BB)
- Actions (RFI, 3-bet, 4-bet, call, fold, raise, limp, squeeze)
- Hand Notations (AJo, AQs, KQo, suited connectors, etc.)
- Concepts (Equity, Pot Odds, MDF, Fold Equity, etc.)
- Strategy Terms (ICM, Bubble, Push-or-Fold, etc.)
